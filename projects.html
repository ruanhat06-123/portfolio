Khulani Accounting Management Services

In this project I created a full website for a local bussiness called Khulani Accounting Management Services. This project was built using HTML, CSS, JavaScript and supabase for data storage. It has a main website consisting of a Home, about, services, plans and pricing, book online, Blog, then then it has a drop down menu where you can select to view vacancies, FAQ's, select to contact, and a members area. Then if you log in with the admin account you get redirected to a admin page where you can add blogs, vacancies and FAQ's, on this page the admin has the option to edit user roles as well.
