Khulani Accounting Management Services

In this project, I created a full website for a local business called Khulani Accounting Management Services. The website was built using HTML, CSS, JavaScript, and Supabase for data storage. It includes pages for Home, About, Services, Plans and Pricing, Book Online, and Blog. It also includes a dropdown menu where users can view vacancies, FAQs, contact information, and a members area. When logged in with an admin account, the user is redirected to an admin page where blogs, vacancies, and FAQs can be added. The admin can also edit user roles.
